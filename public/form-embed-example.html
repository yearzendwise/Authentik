üìù Method 1: HTML Data Attributes (Automatic)

The simplest way to embed a form. Just add the data attributes and the form will automatically load:

<!-- Include the Authentik Forms script --> <script src="https://zendwise.work/js/authentik-forms.js"></script> <!-- Create a container with data attributes --> <div id="my-form-1" data-authentik-form="dc1ebca2-8aeb-4057-ab32-4b4ce248eb04" data-authentik-theme="minimal" data-authentik-show-title="true" data-authentik-show-description="true"> </div>
Note: Replace "YOUR_FORM_ID" with an actual form ID from your Authentik dashboard. This demo uses a placeholder since we don't have a real form set up yet.

‚ö° Method 2: JavaScript API (Programmatic)

For more control, use the JavaScript API to embed forms programmatically:

<!-- Include the script --> <script src="https://zendwise.work/js/authentik-forms.js"></script> <!-- Create a container --> <div id="my-form-2"></div> <script> // Configure the base URL (optional) AuthentikForms.configure({ baseUrl: 'https://zendwise.work' }); // Embed the form const form = AuthentikForms.embed('Hello Form', 'dc1ebca2-8aeb-4057-ab32-4b4ce248eb04', { theme: 'modern', showTitle: true, showDescription: true, onSuccess: function(response) { console.log('Form submitted successfully!', response); alert('Thank you for your submission!'); }, onError: function(error) { console.error('Form submission failed:', error); alert('There was an error submitting the form.'); } }); </script>

‚öôÔ∏è Configuration Options

Customize the form appearance and behavior with these options:

// Available themes themes: ['minimal', 'modern'] // HTML Data Attributes data-authentik-form="dc1ebca2-8aeb-4057-ab32-4b4ce248eb04" // Required: Your form ID data-authentik-theme="minimal" // Optional: Theme (default: minimal) data-authentik-show-title="true" // Optional: Show form title data-authentik-show-description="true" // Optional: Show form description // JavaScript API Options AuthentikForms.embed('container-id', 'dc1ebca2-8aeb-4057-ab32-4b4ce248eb04', { theme: 'minimal', // Theme to use showTitle: true, // Show form title showDescription: true, // Show form description onSuccess: function(response) { }, // Success callback onError: function(error) { } // Error callback });
Important: Make sure your Authentik server is running and accessible. Update the script src URL to match your server's domain.
